# AI Document Assistant

This project is the Day 1 foundation for an AI-powered document assistant.

## What is included

### Backend
- Express server setup
- Clean architecture folders: routes, controllers, services, middleware, utils, config
- Health and welcome endpoints
- CORS, JSON parsing, and dotenv support

### Frontend
- Next.js + TypeScript + Tailwind CSS
- Modern landing page with hero section and future pipeline overview

## Run locally

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
